4．实验方法和步骤（含分析设计）

整体思路 ：

一个多线程复制文件的工具类，在这个多线程复制文件内还有一个守护线程，可以检测目标文件的大小，从而可以计算出占复制目标的百分比。

除了复制文件的工具类之外还需要一个图形界面类，图形界面需要有两个文件选择框和一个现实百分比的label标题

 

首先是复制文件工具类（FileCopyTool）：

拥有两个属性，filepath和copypath分别是源文件和目标文件

 

FileCopyTool需要实现runnable接口才可以实现多线程

public class FileCopyTool implements Runnable

 

编写复制文件的方法：

创建一个文件读取流，和一个文件输出流，分别是源文件和目标文件的

然后创建一个byte数组1024（1kb）的缓冲空间，循环读取写入，复制完成之后输出done，代码如下：

private void Copy() throws IOException {

        FileInputStream fis = new FileInputStream(filepath);

        FileOutputStream fos = new FileOutputStream(copypath);

        byte temp[]=new byte[1024];

        int len;

        while((len=fis.read(temp))>0){

            fos.write(temp,0,len);

        }

        fos.close();

        fis.close();

        System.out.println("done");

        return;

}

因为实现了runnable所以需要完成run方法

在run方法中，调用copy方法，并且创建一个守护线程，死循环读取目标文件的大小并且计算百分比，当百分比等于1（100%）之后跳出循环

代码如下：

@Override

    public void run() {

        Thread shouhu = new Thread(new Runnable() {

            @Override

            public void run() {

                File file = new File(filepath);

                double length = file.length()*1.0;

                File copy = new File(copypath);

                System.out.println(copy.exists());

                while(true){

                    double bfb = copy.length() / length;

                    System.out.println(String.format("%.0f",bfb*100));

                    baifen.setText("当前进度:"+String.format("%.0f",bfb*100)+"%");

                    try {

                        Thread.sleep(20);

                    } catch (InterruptedException e) {

                        e.printStackTrace();

                    }

                    if(bfb>=1){

                    System.out.println("100了 !!");

                    return;

                    }

                }

        shouhu.setDaemon(true);

        shouhu.start();

        try {

            Copy();

        } catch (IOException e) {

            e.printStackTrace();

        }

}
