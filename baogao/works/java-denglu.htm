    }

登录流程设计：

1.       点击按钮之后，执行点击事件

2.       获取用户输入的用户名和密码

3.       从本地用户数据文件读取用户集合

4.       遍历集合，判断有没有存在用户名和密码相同的用户

5.       如果有相同的则登陆成功

6.       如果遍历完没有存在相同的用户名则提示用户不存在

 

登录按钮监听事件：

if(e.getSource()==denglu){

//         获取用户名密码

           String username = this.username_field.getText();

           String password = this.password_field.getText();

           InputStream is = this.getClass().getClassLoader().getResourceAsStream("users.cfg");

           ArrayList<User> users = null;

           try {

              ObjectInputStream ois = new ObjectInputStream(is);

              users = (ArrayList<User>) ois.readObject();

              ois.close();

              is.close();

           } catch (IOException | ClassNotFoundException e1) {

              e1.printStackTrace();

           }

           System.out.println(users);

//         遍历所有的用户

           for(User u : users){

              if(u.username.equals(username) && u.password.equals(password)){

                  JOptionPane.showMessageDialog(null, "登陆成功");

                  return;

              }

           }

           JOptionPane.showMessageDialog(null, "用户不存在");

       }

 

注册流程设计：

1.  用户点击注册按钮

2.  获取用户名和密码

3.  获取本地用户数据集合对象

4.  遍历对象，判断用户是否存在

5.  如果不存在，向集合中追加新的用户

6.  如果存在，则提示用户已存在
