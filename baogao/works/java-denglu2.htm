学生用户类：拥有两个成员，用来保存账号和密码，要实现Serializable接口，这样可以序列化到文件中。

public class Student implements Serializable{

    String Studentname;

    String password;

    public String getStudentname() {

       return Studentname;

    }

    public void setStudentname(String Studentname) {

       this.Studentname = Studentname;

    }

    public String getPassword() {

       return password;

    }

    public void setPassword(String password) {

       this.password = password;

    }

    public Student(String Studentname, String password) {

       super();

       this.Studentname = Studentname;

       this.password = password;

    }

    public Student() {

    }

    @Override

    public String toString() {

       return "Student [Studentname=" + Studentname + ", password=" + password + "]";

    }

   

}

 

用io流读取用户文件，转成对象输入流ois，然后读取对象强转成List集合即可

//  获取所有用户

    public ArrayList<Student> getAllStudent() throws IOException, ClassNotFoundException {

       URL resource = this.getClass().getClassLoader().getResource("Students.cfg");

       ObjectInputStream ois = new ObjectInputStream(new FileInputStream(resource.getPath()));

       ArrayList<Student> allStudents = (ArrayList<Student>) ois.readObject();

       ois.close();

       return allStudents;

    }

外面传进来用户集合ArrayList<Student>，通过fos文件输出流转成对象输出流，保存对象集合，保存完毕后返回true

public boolean saveStudent(ArrayList<Student> list) throws FileNotFoundException, IOException {

       String path = this.getClass().getClassLoader().getResource("Students.cfg").getPath();

       ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream(path));

       oos.writeObject(list);

       return true;

    }

先调用获取所有用户的方法，获取List集合，使用集合的add方法向集合中追加新的用户，然后通过保存用户信息方法写入文件

public boolean addStudent(Student Student) throws IOException, ClassNotFoundException {

       ArrayList<Student> allStudent = getAllStudent();

       allStudent.add(Student);

       return saveStudent(allStudent);

    }

 

 

先获取所有用户然后遍历所有用户，如果用户名相同，则直接返回该用户，如果不存在则返回null

public Student getStudentByStudentname(String Studentname) throws ClassNotFoundException, IOException {

       ArrayList<Student> allStudent = getAllStudent();

       for(Student u : allStudent){

           if(u.Studentname.equals(Studentname)){

              return u;

           }

        }

       return null;

    }
